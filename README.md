Source code for [yasingencnet](https://yasingenc.net), my personal website, built with [Eleventy](https://www.11ty.io).

### Run Project

```
npx eleventy
```

Or build and host locally for local development
```
npx eleventy --serve
```

Or build automatically when a template changes:
```
npx eleventy --watch
```
